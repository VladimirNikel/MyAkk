Подробное описание

В данном файле описан набор функций-обработчиков http-запросов к серверу, а таке вспомогательных функций, необходимых для корректной работы сервера.
Все описанные аргументы функций-обработчиков http-запросов заключены в единственном, передаваемом каждой каждой функции, аргументе request, представляющем
собой словарь, где ключами являются названия аргументов.

◆ add_password()

def psswdmng.views.add_password (   request)

Обработчик http-запроса на добавление основных данных (адрес ресурса, логин и пароль пользователя на ресурсе)

Тип запроса: POST

Аргументы
auth_seqСписок (массив) чисел, представляющий собой байты расшифрованной аутентификационной последовательности, выданной клиентскому приложению
userИмя пользователя, от которого поступает запрос
urlURL-адрес ресурса, на котором осуществляется аутентификация с добавляемым паролем
loginЛогин пользователя, по которому осуществляется идентификация пользователя на ресурсе
passwordДобавляемый пароль, по которому будет осуществляться аутентификация пользователя на ресурсе
Возвращает
Authentification error, если запрос не прошёл аутентификацию
0, если данные успешно дабавлены
Заметки
Если данные, указанные в запросе, не существуют в базе (за исключением имени пользователя), они будут добавлены.

◆ add_user()

def psswdmng.views.add_user (   request)

Обработчик http-запроса на добавление нового пользователя

Тип запроса: POST

Аргументы
auth_seqСписок (массив) чисел, представляющий собой байты расшифрованной аутентификационной последовательности, выданной клиентскому приложению
userИмя пользователя, от которого поступает запрос (в данном случае – пользователь "Система")
usernameИмя добавляемого пользователя
passwordhashХэш пароля добавляемого пользователя
Возвращает
Authentification error, если запрос не прошёл аутентификацию
0, если пользователь добавлен успешно
1, если такое имя пользователя уже зарегистрировано в системе или является зарезервированным
Заметки
Допустимы только имена, под которыми ещё не были зарегистрированы пользователи в системе. Также запрещено имя sys, т.к. оно зарезервировано для нужд системы.

passwordhashХэш пароля пользователя, являющийся аутентификатором
Возвращает
Authentification error, если запрос не прошёл аутентификацию
0, если пользователь успешно прошёл аутентификацию
1, аутентификация пользователя провалена
